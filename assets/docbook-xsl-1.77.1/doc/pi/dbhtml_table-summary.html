dbhtml_table-summary

dbhtml_table-summary — Specifies summary for CALS table, variablelist, segmentedlist, or qandaset output

Synopsis

<?dbhtml table-summary="text"?>

Description

Use the <?dbhtml table-summary?> PI as a child of a CALS table, variablelist, segmentedlist, or qandaset to specify the text for the HTML summary attribute in the output HTML table.

Parameters

table-summary="text"

Specifies the summary text (zero or more characters)
