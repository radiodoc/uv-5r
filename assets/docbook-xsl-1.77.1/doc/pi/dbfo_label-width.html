dbfo_label-width

dbfo_label-width — Specifies the label width for a qandaset, itemizedlist, orderedlist or calloutlist

Synopsis

<?dbfo label-width="width"?>

Description

Use the <?dbfo label-width?> PI as a child of a qandaset, itemizedlist, orderedlist, or calloutlist to specify the width of labels.

Parameters

label-width="width"

Specifies the label width (including units)
