refentry.generate.title

refentry.generate.title — Output title before refnames?

Synopsis

<xsl:param name="refentry.generate.title" select="0"></xsl:param>

Description

If non-zero, the reference page title or first name is output before the list of refnames. This parameter and refentry.generate.name are mutually exclusive. This means that if you change this parameter to non-zero, you should set refentry.generate.name to zero unless you want get quite strange output.
