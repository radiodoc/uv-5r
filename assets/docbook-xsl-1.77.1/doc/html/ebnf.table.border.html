ebnf.table.border

ebnf.table.border — Selects border on EBNF tables

Synopsis

<xsl:param name="ebnf.table.border" select="1"></xsl:param>

Description

Selects the border on EBNF tables. If non-zero, the tables have borders, otherwise they don't.
