qandadiv.autolabel

qandadiv.autolabel — Are divisions in QAndASets enumerated?

Synopsis

<xsl:param name="qandadiv.autolabel" select="1"></xsl:param>

Description

If non-zero, unlabeled qandadivs will be enumerated.
