webhelp.include.search.tab

webhelp.include.search.tab — Should the webhelp output include a Search tab?

Synopsis

<xsl:param name="webhelp.include.search.tab">true</xsl:param>

Description

Set this parameter to 0 to suppress the search tab from webhelp output.
