htmlhelp.hhc.folders.instead.books

htmlhelp.hhc.folders.instead.books — Use folder icons in ToC (instead of book icons)?

Synopsis

<xsl:param name="htmlhelp.hhc.folders.instead.books" select="1"></xsl:param>

Description

Set to non-zero for folder-like icons or zero for book-like icons in the ToC. If you want to use folder-like icons, you must switch off the binary ToC using htmlhelp.hhc.binary.
